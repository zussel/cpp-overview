Lambda Funktionen

Lambda Funktionen sind anonyme Funktionen die ihre gebundenen Variablen konservieren

Vor Lambda Funktion gab es Funktionsobjekte


struct adder {
  adder(int x) : x(x) {}
  int operator()(int y) const { return x + y; }

  int x;
};

// create an instance of the functor class
adder add42(42);

// and "call" it
int i = add42(8);
