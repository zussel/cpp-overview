Initialisierungslisten

Anhand von Initialisierungslisten können Klassen und Container einheitlich initialisiert werden. 


std::vector<int> int_vector = {1,2,3,4,5,6};

for (int i : {1,2,3,4,5,6}) {
    std::cout << i << "\n";
}

Beispiel einer Klasse, die eine Initialisierungsliste per Konstruktor zur Verfügung stellt. 


struct columns
{
    std::vector<std::string> column_vector;

    columns(std::initializer_list<std::string> names)
