Template Meta Programming

TMP erlaubt es Berechnungen, Entscheidungen oder Validierungen schon während der Compilezeit vorzunehmen

Weiterhin wurden Templates um Variadic Templates erweitert, eine beliebige Anzahl von Template Parametern zu beschreiben.

Variadic Templates: adder Funktion


template<typename T>
T adder(T v) {
  return v;
}

template<typename T, typename... Args>
T adder(T first, Args... args) {
  return first + adder(args...);
}

long sum = adder(1, 2, 3, 8, 7);

Berechnungen: Fibonacci


template<int n>
struct fibonacci
{
  static constexpr int value =
    fibonacci<n-1>::value + fibonacci<n-2>::value;
};
template<>
struct fibonacci<0>
{
  static constexpr int value = 0;
};
template<>
struct fibonacci<1>
{
  static constexpr int value = 1;
};
